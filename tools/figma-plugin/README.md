# PSD to Figma Importer Plugin

PSD エクスポーターが出力した `.figma.json` を Figma に読み込むプラグインです。

## インストール

1. Figma Desktop を開く
2. **Menu → Plugins → Development → Import plugin from manifest...**
3. このフォルダの `manifest.json` を選択

## 使い方

1. Figma でファイルを開く
2. **Plugins → Development → PSD to Figma Importer** を実行
3. `.figma.json` ファイルを選択して **Import**
